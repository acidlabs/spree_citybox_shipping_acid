Installation
------------

Add spree_chilean_shipping to your Gemfile:

```ruby
gem 'spree_chilean_shipping'
```

Bundle your dependencies and run the installation generator:

```shell
bundle
bundle exec rails g spree_chilean_shipping:install
```
